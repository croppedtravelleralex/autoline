## 1. 引言 (Introduction)

### 1.1 项目背景
本项目旨在为微光夜视仪连续生产线开发一套智能监控系统。生产线分为 **阴极线 (Cathode Line)** 和 **阳极线 (Anode Line)**，各工序腔室由真空插板阀隔离。系统需实现对生产全流程的实时数据采集、远程控制、工艺流程管理及异常互锁保护，确保生产过程的数据真实性与受控性。

### 1.2 适用范围
本系统适用于连续生产线的中控室及现场可视终端，用户涵盖操作员、工艺工程师、设备工程师及系统管理员。

---

## 2. 系统概述 (System Overview)

### 2.1 线体结构
系统核心监控两条独立运行但逻辑关联的真空线体：
*   **阳极线 (Anode)**: 进样仓 $\to$ 烘烤仓 $\to$ 冷却仓 $\to$ 清刷仓 $\to$ 对接仓 $\to$ 铟封仓 $\to$ 出样仓
*   **阴极线 (Cathode)**: 进样仓 $\to$ 烘烤仓 $\to$ 生长仓 (激活) $\to$ 出样仓

### 2.2 传输逻辑
*   **物理隔离**: 腔室间通过 **插板阀 (Gate Valve)** 隔离。
*   **传输条件**: 仅当相邻腔室 **压差合格** 且 **阀门开启** 时，方可允许小车传输。
*   **进样流程**: 大气装载 $\to$ 关闭舱门 $\to$ 粗抽 $\to$ 开启前级阀 $\to$ 高真空 $\to$ 打开插板阀传输。

## 3. 功能需求 (Functional Requirements)

### 3.1 实时监控与控制 (Real-time Monitoring)
*   **可视化界面 (HMI)**:
    *   采用 **工业赛博风格**，深蓝色背景，高对比度信息显示。
    *   **拟物化状态**: 加热红光特效，真空度深浅视觉反馈。
    *   **扁平化卡片**: 显示腔体温度、真空度 (Pa)、阀门/泵状态 (绿/灰/黄/红四色逻辑)。
*   **设备控制**:
    *   **远程操作**: 控制插板阀、前级阀、粗抽阀、放气阀及各类真空泵。
    *   **安全操作**: 关键操作（如手动开阀）需 **长按3秒** 或 **二次确认**。

### 3.2 智能仿真与互锁 (Simulation & Interlocks)
*   **后端仿真引擎**:
    *   **温度模拟**: 模拟升温/降温曲线（如阳极烘烤仓 10h 升温至 390℃）。
    *   **真空模拟**: 模拟抽气速率与泄露率，计算实时真空度。
*   **硬性互锁 (Safety Interlock)**:
    *   **真空互锁**: 相邻腔压差 $> 1$ 级 ($10^{-3}$ vs $10^{-5}$ Pa) 时，**物理禁用** 插板阀开启。
    *   **放气互锁**: 插板阀开启时，**禁止操作** 放气阀。
    *   **冲泵保护**: 若监测到高压差下开启分子泵，立即触发 **黄色闪烁警告** 并记录审计日志。

### 3.3 工艺配方管理 (Recipe Management)
*   **权限分级**:
    *   **Draft**: 工程师编辑。
    *   **Review**: 主管审核。
    *   **Approved**: 部长签核（仅批准后的配方可被操作员加载）。
*   **版本控制**: 严禁覆盖旧版本，必须另存为新版本号 (v1.0 $\to$ v1.1)。
*   **参数比对**: 自动高亮显示不同版本间的参数差异。

### 3.4 生产数据追溯 (Traceability)
*   **一管一档 (EBR)**: 每个产品生成独立电子批记录。
*   **数据记录**:
    *   关键工序（烘烤/激活）：采样率 $\ge 1 \text{Hz}$。
    *   非关键工序：采样率 $\ge 0.1 \text{Hz}$。
*   **数据完整性**: 原始数据加密存储 (FDA 21 CFR Part 11 合规)。

## 4. 详细工艺规格 (Process Specifications)

### 4.1 阳极线 (Anode) Key Specs
| 工序 | 时长 | 温度 (内温) | 关键动作/要求 |
| :--- | :--- | :--- | :--- |
| **A-01 进样** | 30m | 常温 | 预抽至 $1-5 \text{Pa}$ |
| **A-02 烘烤** | 10h+ | $390^\circ\text{C}$ | 升温10h $\to$ 降温5h |
| **A-03 冷却** | >12h | < 常温 | 确保完全冷却防止碎裂 |
| **A-04 清刷** | 5.5h | 常温 | 高压 $5\text{kV}$, 电流 $300\mu\text{A}$ |
| **A-05 对接** | 4h | $90-100^\circ\text{C}$ | 预热除气 |
| **A-06 铟封** | 4h | < $110^\circ\text{C}$ | **联动**: 需等待阴极生长完成 |

### 4.2 阴极线 (Cathode) Key Specs
| 工序 | 时长 | 温度 (内温) | 关键动作/要求 |
| :--- | :--- | :--- | :--- |
| **C-01 进样** | 30m | 常温 | N2 置换 3 次 |
| **C-02 烘烤** | 10-24h | $400-410^\circ\text{C}$ | 必须显示“已除气”绿灯 |
| **C-03 生长** | >4h | $230 \to 110^\circ\text{C}$ | 初始5h降温; 铯/氧源控制 |

## 5. 接口与系统指标
*   **响应时间**: 状态刷新延迟 $< 1000 \text{ms}$ (Polling Interval)。
*   **并发支持**: 支持多客户端同时在线查看（Web Architecture）。
*   **数据导出**: 支持 `.csv`, `.xlsx` 格式导出，支持月度趋势对比。
