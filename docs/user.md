# 用户权限与账户定义 (User Permissions & Accounts)

本文档记录系统中的用户角色、权限定义以及调试阶段的预设账户。

## 1. 角色定义 (Roles)

系统包含以下两种主要角色：

### 1.1 超级管理员 (Super Admin)
*   **权限等级**: 最高
*   **功能权限**:
    *   查看所有页面和数据 (View All)
    *   **编辑线体 (Edit Line)**: 允许添加、删除、移动腔体，修改线体配置。
    *   **设备控制**: 允许控制阀门、泵、小车运动等。
    *   用户管理: 允许添加/删除用户。

### 1.2 观察员 (Observer)
*   **权限等级**: 只读
*   **功能权限**:
    *   查看部分或全部页面 (View Only)
    *   **禁止编辑**: 无法进入或保存线体编辑。
    *   **禁止控制**: 无法操作阀门、泵或移动小车。
    *   **禁止管理**: 无法修改系统设置或用户。

## 3. 快速登录 (Quick Login - Debugging)

在调试阶段，界面顶部的用户图标将提供快捷切换入口，无需每次输入密码即可快速切换身份以测试权限控制。
